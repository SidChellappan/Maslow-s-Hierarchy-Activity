# Build Maslow's Pyramid

An age-appropriate interactive web demo for 4th graders: **Build Maslow's Pyramid: What People Need to Feel Safe and Human**.

Students build a five-level pyramid of human needs, explore gentle non-graphic scenarios about how injustice strips away needs, then rebuild the pyramid with upstander actions.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The app is a single-page React + TypeScript + Tailwind CSS project with no backend and no paid APIs.
